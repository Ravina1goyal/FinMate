# FinMate 💰

FinMate is a full-stack personal finance management application built with React Native, TypeScript, Node.js, Express, and MongoDB.

The application helps users track expenses, manage bills, monitor subscriptions, analyze spending patterns, and improve financial planning through a modern mobile experience.

## Features

* Expense Tracking
* Income Management
* Budget Planning
* Bill Reminders
* Subscription Management
* Spending Analytics
* Push Notifications
* Offline Support
* Secure Authentication

## Tech Stack

### Frontend

* React Native
* TypeScript
* React Navigation
* Redux Toolkit
* AsyncStorage

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication
